Sube tu Web a GitHub Pages

La guía definitiva para desplegar tu portafolio o proyecto gratis en minutos.

Comenzar Ahora

¿Cómo Subir tu Página?

1. Crea un Repositorio

Ve a GitHub, crea un nuevo repositorio público con el nombre que desees (ej: mi-web).

2. Prepara tus Archivos

Asegúrate de que tu archivo principal se llame index.html. Es obligatorio para que funcione.

3. Sube el Código

Usa git push o sube los archivos manualmente mediante la interfaz web de "Upload files".

4. Activa Pages

En "Settings" > "Pages", selecciona la rama main o master y guarda. ¡Listo!
